# 本章概览：从零到上线

这一章带你完整走一遍项目开发流程：从配置环境，到写代码，再到部署上线。

---

## 你会学到什么？

学完这章，你能独立完成：

- 配置好 Claude Code 开发环境
- 让 AI 帮你写后端代码
- 让 AI 帮你写前端代码
- 把项目部署到线上，让全世界都能访问

---

## 整体流程

```
环境准备 → 需求设计 → 后端开发 → 前端开发 → 部署上线
   ↓          ↓          ↓          ↓          ↓
 3篇文章    1篇文章    1篇文章    1篇文章    1篇文章
```

## 技术栈说明

这章用的技术栈：

| 类型 | 技术 | 为什么选它 |
|------|------|-----------|
| 前端 | Next.js + Tailwind CSS | 主流、好学、部署方便 |
| 后端 | FastAPI + PostgreSQL | Python 写的，简单易懂 |
| 数据库 | Supabase | 免费、功能全、自带用户认证 |
| 部署 | Vercel + Railway + Cloudflare | 免费额度够用、一键部署 |

## 学之前需要会什么？

**不需要会：**
- 不需要会写代码（Claude 帮你写）
- 不需要懂服务器（平台帮你管）
- 不需要会数据库（跟着教程配就行）

**需要会：**
- 会用电脑打字
- 会注册账号
- 能看懂简单的英文单词

---

## 遇到问题怎么办？

1. **先问 Claude** — 把报错信息贴给它，让它帮你排查
2. **查名词手册** — 不懂的词去《08_名词手册》查
3. **搜一下** — 报错信息 + 关键词，Google 一下
4. **问社群** — 实在搞不定，去社群问问其他人
